Interaktive Python-Umgebung mit PyScript

Dieser Kurs enthält eine interaktive Webversion mit PyScript-Integration, die es ermöglicht, Python-Code direkt im Browser auszuführen.

Vorteile der PyScript-Integration:

  • Python-Code direkt im Browser ausführen, ohne Installation
  • Interaktive Beispiele und Übungen mit sofortigem Feedback
  • Nahtlose Integration von Erklärungen und ausführbarem Code
  • Zugänglich von jedem Gerät mit einem modernen Webbrowser

Voraussetzungen

  • Python 3.8 oder höher
  • Git

Lokales Testen der Website

Um die Website lokal zu testen:

cd docs
python -m http.server 4000

Die Website ist dann unter http://localhost:4000 verfügbar.
